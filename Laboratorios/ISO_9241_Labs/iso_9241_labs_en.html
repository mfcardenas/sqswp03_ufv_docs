ISO 9241 is a multi-part standard that covers the ergonomics of human-system interaction. It provides guidelines and requirements for the design of usable, accessible, and ergonomic interactive systems.

This laboratory series contains 10 practical laboratories designed to explore and apply ISO 9241 principles in real-world scenarios.

Laboratory List

Lab 1Introduction to ISO 9241

Objective

Create a simple web application that demonstrates the basic principles of ISO 9241, particularly ISO 9241-11 (Guidance on usability).

Requirements

  • Design a login form with fields for username and password
  • Ensure the form follows usability principles: effectiveness, efficiency, and satisfaction
  • Include error handling for invalid inputs
  • Provide feedback to the user after submission
  • Write a Python script to simulate user interaction and evaluate the form's usability

Key Tasks

  • Create an HTML file for the login form
  • Add CSS for styling to improve visual ergonomics
  • Implement JavaScript for client-side validation
  • Write a Python script using Selenium to automate user testing and measure task completion time

Expected Output

A functional web page with a usable login form and a Python script that runs automated tests and reports usability metrics.

Lab 2Usability Principles

Objective

Analyze and improve a poorly designed web form that violates several ISO 9241 usability principles.

Requirements

  • Identify usability violations in the provided code
  • Redesign the form to comply with ISO 9241-11 principles
  • Implement improvements in HTML, CSS, and JavaScript
  • Create a Python script to compare the usability of the original and improved versions
  • Document the changes and their impact on usability metrics

Key Tasks

  • Analyze the original form code for usability issues
  • Redesign the form with better layout, feedback, and accessibility
  • Add client-side validation and error handling
  • Write a Python script using Selenium to measure task completion times for both versions
  • Compare the results and explain improvements

Expected Output

Improved HTML, CSS, and JavaScript code, a Python testing script, and documentation of usability improvements.

Lab 3Ergonomic Requirements

Objective

Design a web-based text editor that incorporates ergonomic principles from ISO 9241-400 series (Physical input devices).

Requirements

  • Implement keyboard shortcuts for common actions
  • Design an interface that reduces mouse usage
  • Include adjustable font sizes and contrast settings
  • Add features to prevent repetitive strain injuries
  • Create a Python script to analyze user interaction patterns
  • Ensure compliance with ergonomic guidelines for visual display units

Key Tasks

  • Create an HTML editor with ergonomic features
  • Implement keyboard navigation and shortcuts
  • Add customizable display settings
  • Write JavaScript for ergonomic interactions
  • Develop a Python script to log and analyze user behavior
  • Document ergonomic improvements and their benefits

Expected Output

A functional text editor with ergonomic features, Python analysis script, and documentation of ergonomic compliance.

Lab 4User-Centered Design

Objective

Apply the user-centered design (UCD) process from ISO 9241-210 to redesign an existing e-commerce checkout form.

Requirements

  • Analyze the current checkout form for usability issues
  • Create user personas and scenarios
  • Design wireframes and prototypes
  • Conduct usability testing
  • Implement improvements based on user feedback
  • Create a Python script to simulate user testing and analyze results

Key Tasks

  • Document current usability problems
  • Develop user personas and use cases
  • Create HTML/CSS/JS prototypes of improved design
  • Implement A/B testing functionality
  • Write Python scripts for automated user testing
  • Analyze and document improvements

Expected Output

User personas and scenarios, multiple design iterations, functional prototype with testing capabilities, and Python analysis scripts.

Lab 5Accessibility Standards

Objective

Design and implement an accessible web application that complies with WCAG 2.1 guidelines and ISO 9241 accessibility requirements.

Requirements

  • Implement WCAG 2.1 Level AA compliance
  • Add support for screen readers and assistive technologies
  • Ensure keyboard-only navigation
  • Provide alternative text for all images and media
  • Implement high contrast mode and adjustable text sizes
  • Create a Python script to test accessibility compliance

Key Tasks

  • Design an accessible web page with proper semantic HTML
  • Implement ARIA attributes and roles
  • Add keyboard event handlers and focus management
  • Create CSS for high contrast and scalable text
  • Write JavaScript for dynamic accessibility features
  • Develop Python script using accessibility testing tools

Expected Output

WCAG compliant web page, Python accessibility testing script, and documentation of accessibility features.
