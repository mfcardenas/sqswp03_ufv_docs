ISO Standards Live Quiz ‚Äì Multiplayer session control

ISO Standards Live Quiz ‚Äì Secure host/player experience

üéÆ Launch external experience

A classroom-ready platform inspired by live quiz shows that links a password-protected teacher console with a distraction-free player view. Instructors configure the session, pick between LLM-generated or curated ISO question packs, and trigger every round while students answer from their phones.

Roles and guided flow

  • Teacher console: Password gate, unique session code, and controls to start/close rounds, run the timer, and monitor rankings in real time.
  • Student view: Minimal UI with session code, live status, visual timer, and explicit confirmation before locking responses.
  • Language sync: The host preference is stored in-browser and automatically mirrored across every connected player.
  • Default host password: teachISO! (can be configured inside config.js).

Session-level configuration

  • Custom ISO context: Teachers set the title, storyline, and focus standards (ISO 9241, ISO/IEC/IEEE 29148, ISO/IEC 25010:2023).
  • Catalog or LLM: Toggle between vetted banks or fresh prompts generated through Ollama (gpt-oss) with automatic fallback.
  • Timing and difficulty: Control the number of questions, per-round timers, and whether the run is monolingual or bilingual.

Student-side experience

  • Immediate feedback: States for waiting, live question, and partial results after every closure.
  • Classroom safety: The player UI strips navigation links and only accepts input while the round is open.
  • Full-device support: Responsive layout validated on phones and desktops with baseline accessibility baked in.

Technology and guarantees

  • Node.js + Express + Socket.IO stack with strict role separation and automatic session cleanup.
  • Lightweight localStorage persistence to remember host language and preference toggles.
  • ISO-focused question catalog plus fallback routines whenever the LLM endpoint is unavailable.
  • QA completed in November 2025 for hybrid and in-person cohorts.

Recommended methodology

üéØ Always run with teacher facilitation

  • Teachers prepare the scenario and validate the question source before sharing the session code.
  • Each round includes a quick debrief on the ISO clause involved plus metric review.
  • Close with the live ranking and actionable improvements tied to the real project.
